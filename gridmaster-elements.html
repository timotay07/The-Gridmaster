Getting Started

First things first. Clone or download this project to your desktop. Open the index page in a browser. Blow up everything between the opening <body> tag and the first <script> tag. Now you have a clean boilerplate, no need to hookup a new page and copy paste stylsheets and scripts, it's already there.

So you have your blank project. Now what? Determine your Magic Numbers.

Your first Magic Nmber: <p> font-size. The second Magic Number: <p> line-height.

The Gridmaster starts out with the Magic Numbers 16(default browser body font-size) and 24(the product of 1.5, common line-height/font-size ratio of modern web typgraphy).

The Control Panel

The control panel is located in the bottom left corner of the screen. From the control panel you can increment the line-height and number of columns. Additionally, you can set the page or container width. If you want to preview your layout without the grid and baseline, simply toggle Gridmaster with the "Grid Off" button.

Setting Defaults

Incrementing line-height, columns and page width is fine when trying to determine your project settings, but once you have decided on the parameters, set the Gridmaster defaults in the gridmaster.js file. Near the top of the document, you will find three functions that will set the default line-height, number of columns, and page width.

  • $("input[name=lineHeight]").val();
  • $("input[name=numberColumns]").val();
  • $("input[name=containerWidth]").val();

Simply plug-in your values like so:

  • $("input[name=lineHeight]").val(20);
  • $("input[name=numberColumns]").val(10);
  • $("input[name=containerWidth]").val(960);
